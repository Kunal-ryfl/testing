# Java Multi-Project Repository

This repository contains **two separate Java projects**, each demonstrating different concepts in Java programming. Both projects exist inside this root folder as independent modules.

---

## 1. Producer–Consumer Synchronization

This project showcases the classic **Producer–Consumer problem** using multiple Java concurrency mechanisms.

### ✅ Key Concepts
- Multi-threading
- `wait()` and `notifyAll()`
- `BlockingQueue`
- Concurrent data structures
- Thread coordination and synchronization

---

## 2. Sales Analysis System

This project loads CSV sales data and performs multiple analytics operations using Java Streams.

### ✅ Features
- Compute total revenue
- Group quantity sold by category
- Calculate revenue by product
- Identify the top-selling product
- Compute average sale value
- Comprehensive unit tests using JUnit
